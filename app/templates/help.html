File List Service Help

Welcome to the File List Service. This service provides a paginated list of files and directories in a given directory.

Usage

To use this service, simply navigate to the URL of the directory you want to list:

http://localhost:5000/files/PATH_TO_DIRECTORY

You can also specify additional query parameters:

  • page_num: the page number to display (default: 1)
  • page_size: the number of items per page (default: 10)
  • search_string: a string to search for in the file paths (default: None)
  • json: if true, returns a JSON response instead of an HTML response (default: false)
